# Claude Projects — Setup Guide

Create a persistent AI workspace that remembers your context, files, and instructions across every conversation.

## What It Is
Claude Projects lets you give Claude a permanent system prompt, upload reference documents, and have all your conversations in that project share the same context — without re-explaining yourself every time.

## Setup (Requires Claude Pro — $20/mo)

1. Go to [claude.ai](https://claude.ai) and sign in (or create an account)
2. In the left sidebar, click **+ New Project**
3. Give your project a name (e.g. "Content Strategy", "Trading Research", "Client Work")
4. Click **Project Instructions** → write a system prompt:
   - Who Claude is in this context
   - What it should and shouldn't do
   - Any standing preferences or rules
   - Example: *"You are my content assistant for @opti.algo. Always write in a concise, no-hype tone. Never use emojis unless asked. Audience is systematic traders."*
5. Click **Add Content** to upload files:
   - PDFs, text files, markdown — anything you want Claude to reference
   - Good examples: brand guidelines, research documents, SOPs, previous work
   - Up to 200KB per file, multiple files allowed
6. Start a conversation inside the project — Claude will follow your instructions and reference your files automatically in every chat

## What Makes It Powerful
- **No re-explaining**: Claude remembers your instructions and documents in every new conversation
- **Multiple projects**: Set up separate projects for different contexts (work, creative, personal)
- **File references**: Ask Claude to "use the brand guide I uploaded" and it pulls from your files directly

## Tips
- Write project instructions like you're onboarding a smart new employee — the more specific, the better
- Upload your most-referenced documents (not every file you own)
- Projects don't share memory between each other — each is isolated

## Free vs Pro
- **Free**: No Projects feature
- **Pro ($20/mo)**: Full Projects access, longer context, priority access
